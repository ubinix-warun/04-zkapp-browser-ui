

# 04-ZkApp-Brower-UI

```
$ zk project 04-zkapp-browser-ui --ui next

```

## Build Contract

```
$ cd contracts/
$ npm run build

```

## Build UI

```
First Terminal
$ npm run dev

Second Terminal
$ npm run ts-watch

```

## Deploy gh-page

```
npm run deploy

```
