

# 04-ZKAPP-BROWSER-UI

```
zk project 04-zkapp-browser-ui --ui next

```

## Build Contract

```
cd contracts/
npm run build

```

## Build UI

```
npm run dev

npm run ts-watch

```

## Deploy gh-page

```
npm run deploy

```
